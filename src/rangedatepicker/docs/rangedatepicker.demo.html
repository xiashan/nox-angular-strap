RangeDatepickers datepicker.js

mgcrea.ngStrap.rangedatepicker

Examples

Add range datepicker functionality with any form text input.

Plugin dependency

RangeDatepickers require the tooltip module and dateParser helper module to be loaded.

These docs currently use bootstrap-additions for styling purposes.

Support for locales

This module leverages the $locale service. You just have to load the proper i18n file to seamlessly translate your datepickers.
